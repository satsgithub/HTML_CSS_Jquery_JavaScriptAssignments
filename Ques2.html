Tag Description Example
<b></b> Bold text This text is bold.
<strong></strong> Important text This text is important.
<i></i> Italic text This text is italic.
<em></em> Emphasized text This text is emphasized.
<mark></mark> Marked text This text is marked.
<small></small> Smaller text This text is small.

<ins></ins> Inserted text This text is inserted.
<sub></sub> Subscript text This text is subscript.
<sup></sup> Superscript text This text is superscript.
<u></u> Underlined text This text is underlined.
<s></s> Strikethrough text This text is strikethrough.
<q></q> Inline quote This is an inline quote.
<blockquote></blockquote> Block quote
This is a block quote. It can span multiple lines.
<code></code> Inline code This is inline code.
<pre></pre> Preformatted text 
This is preformatted text. It preserves spaces and line breaks.
